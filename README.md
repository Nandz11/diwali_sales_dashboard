# 🪔 Diwali Sales Dashboard

An interactive dashboard analyzing customer buying behavior during Diwali festival.

## Live Demo
(add Streamlit link here after deployment)

## What it does
- Analyzes 11,239 customer transactions during Diwali
- Interactive filters by Gender, Age Group and Zone
- KPI metrics — Total Revenue, Customers, Orders, Avg Spend
- Sales breakdown by Gender, Age, State, Category and Occupation

## Key Insights
- Women spend 2x more than men during Diwali (₹74M vs ₹31M)
- Age group 26-35 are the biggest spenders
- Uttar Pradesh is the top shopping state
- Food is the #1 product category during Diwali

## Tech Stack
- Python, Pandas
- Plotly (interactive charts)
- Streamlit (dashboard)

## How to run
pip install -r requirements.txt
streamlit run app.py
